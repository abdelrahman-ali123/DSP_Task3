Feature Extraction using MFCC

The first step in voice recognition processing is to a developed system to extract computed features of human voice.
We used MFCC (Mel Frequency Cepstral Coefficient) technique to extract these voice features from persons. MFCC considers the human voice pitch in the form of frequencies and scale them on the Mel scale, these extracted features are unique to others.
After Feature Extraction process, the system undergoes Training and Matching phases.

ML Model - GMM

In training process, the extracted features are trained using the Gaussian Mixture Modeling.
The Gaussian Mixture Model is a generative model that assumes each data cluster as a different Gaussian distribution. Then it will tell, based on probability, out of which distribution that data point came out and finally used to store in a database.
In matching process, the system authenticates the registered person by matching the real time voice sample to the stored voice samples in the database.
